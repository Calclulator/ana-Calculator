WIND SHEAR Method
GFS-VWS (Vertical Wind Shear)

GFS の風データ (u, v) から巡航 FL ± 2000ft の風ベクトル差を計算し、1000ft あたりの風シアー強度を kt/1000ft で表示。

ANA WSCP の VWS 公式 (NAV LOG Appendix 4) と整合するよう 1 便分の実測値で calibration 済 (回帰: gfs ≈ 0.56 × navlog + 3.08)。

カテゴリ: Smooth (0-3) / Light- (4-5) / Light (6-7) / Light+ (8-9) / Moderate (10-12) / Severe (13+) kt/1000ft

注意: Light+ 以上の閾値は外挿で信頼度低、参考値として利用。

※ 閾値の暫定調整日: 2026-05-07。今後のデータ蓄積で再調整予定。

Ellrod TI1 (Turbulence Index 1)

Ellrod & Knapp (1992) による晴天乱気流指標。

TI1 = VWS × |DEF| (DEF = total deformation = √(stretching² + shearing²))

ジェット気流近傍の CAT 検出に使われる。値が大きいほど CAT 確率高。

GFS データから直接計算 (calibration 不要)。

Ellrod TI2 (Turbulence Index 2)

TI1 を拡張し、収束 (convergence) の効果を加算した指標。

TI2 = VWS × (|DEF| + CVG)

ana-Calc では CVG ≤ 0 (発散側) のとき clip するため、発散卓越領域では TI1 = TI2 となる仕様。

共通の注意

GFS は 0.25° 解像度で、局所的な対流乱気流や山岳波は表現できない。SIGMET / PIREP / 機内レーダーと併用して総合判断すること。
