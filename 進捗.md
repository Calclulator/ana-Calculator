## 重要な設計決定事項 (本日確定)
- **S/U 時刻 = Z (UTC) 入力固定** (NAVLOG が Z 表記、現場で Z 基準で考えるため)
- **FDP/Block table 引きは順応地域 LCL** (OM 8-5-1 ①注1)、内部で Z → LCL 換算
- **順応地域 offset デフォ +9 (JST)**、ユーザー変更可
- **table 値は内部分単位** (60 倍丸め回避、精度保持)
- **マルチプル + Class 2/3 → Block 12h59m** (FRM rule)
- **ダブル + Class 2/3 → Block = min(FDP, 17h)**
- **NAVLOG 自動入力 < 手動編集** (手動優先、Clear で復活)
